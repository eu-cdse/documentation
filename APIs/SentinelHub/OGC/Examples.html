Examples of OGC API

Below are examples for all our OGC APIs. To run the examples yourself, replace <INSTANCE_ID> in the URLs with your own configuration instance id and paste the url in any web browser. Your configuration must be based on the "Simple WMS template", which can be found when you create new configuration in the Dashboard in "Configuration Utility" tab.

If you want to check interactive example use this link.

WMS #1

URL

https://sh.dataspace.copernicus.eu/ogc/wms/<INSTANCE_ID>?REQUEST=GetMap&BBOX=3238005,5039853,3244050,5045897&LAYERS=NATURAL-COLOR&MAXCC=20&WIDTH=320&HEIGHT=320&FORMAT=image/jpeg&TIME=2018-03-29/2018-05-29

Parameters

Parameters Options
LAYERS NATURAL-COLOR
FORMAT image/jpeg
MAXCC 20
WIDTH 320
HEIGHT 320
TIME 2018-03-29/2018-05-29

Result
